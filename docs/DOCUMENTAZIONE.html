1. Panoramica Generale

Telaio2D Web è un'applicazione web interattiva per l'analisi statica lineare di strutture intelaiate 2D. L'applicazione permette di generare automaticamente la geometria di telai multipiano e multicampata, modificare i parametri strutturali tramite un'interfaccia intuitiva, eseguire l'analisi agli elementi finiti utilizzando il motore di calcolo OpenSeesPy, e visualizzare i risultati comprensivi di schema strutturale e deformata.

Pubblico target: Ingegneri strutturali, progettisti, studenti di ingegneria civile.
Scopo: Analisi lineare elastica di telai piani ai sensi delle NTC 2018.

2. Funzionalità Principali

🔧 Generatore Telaio

Generazione automatica di telai 2D multipiano e multicampata specificando distanze campate e altezze piani.

📊 Editor XLSX

Modifica interattiva di tutti i dati strutturali (nodi, elementi, proprietà, carichi, vincoli) tramite tabelle editabili.

⚙️ Solver OpenSeesPy

Analisi statica lineare con elementi beam elastici, supporto per carichi distribuiti (uniformi e trapezoidali).

📈 Visualizzazione

Render schema strutturale con vincoli, carichi e deformata scalabile. Visualizzazione immediata dei risultati.

📁 Export Risultati

Esportazione del modello e dei risultati in formato Excel (.xlsx) per documentazione e ulteriori elaborazioni.

✓ Validazione

Validazione automatica del modello strutturale prima dell'analisi, con segnalazione di errori e incoerenze.

6. Metodologia di Calcolo

6.1 Formulazione Elemento Beam Elastico

Matrice di rigidità locale elemento beam 2D (6 DOF)
K = [ EA/L    0           0           -EA/L     0           0;
0    12EI/L³    6EI/L²     0    -12EI/L³    6EI/L²;
0    6EI/L²    4EI/L       0    -6EI/L²    2EI/L;
-EA/L    0           0        EA/L     0           0;
0    -12EI/L³ -6EI/L²     0    12EI/L³   -6EI/L²;
0    6EI/L²    2EI/L       0    -6EI/L²    4EI/L ]

6.2 Sistema Globale

Trasformazione coordinate (geometric transformation)
K_gлоб = Tᵀ · K_lосal · T

dove T è la matrice di trasformazione che converte le grandezze dal sistema locale all sistema globale.

6.3 Sistema Equazioni

Equazione equilibrio lineare
K · u = F

K = matrice di rigidezza globale assemblata (6×n nodi)
u = vettore spostamenti nodali incogniti
F = vettore forze nodali equivalenti (carichi + pesi propri)

6.4 Carico Trapezoidale

Approssimazione carico distribuito trapezoidale
q(y) = q₀ + (q₁ - q₀) · ξ    con ξ ∈ [0,1]

Il carico trapezoidale viene discretizzato in n segmenti uniformi (default: 10) per la compatibilità con OpenSeesPy.
